# SimpleCalculator

## Example
```html
<script src="https://raw.githubusercontent.com/notshekhar/calculate.js/main/calculate.js"></script>
<script>
  calculate(1, 1, "+") // 1 + 1 = 2
  calculate(1, 1, "-") // 1 - 1 = 0
  calculate(1, 1, "/") // 1 / 1 = 1
  calculate(1, 1, "%") // 1 % 1 = 0
</script>
```
